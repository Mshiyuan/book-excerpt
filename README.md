# 书摘

作者：时鸢

在聊天界面对消息划线、写想法，并将摘录与想法导出为图片卡片；支持把不连续的划线合并成一条书摘。
所有数据（设置、笔记本）存于浏览器本地 localStorage，不上传任何服务器。

## 安装

SillyTavern → 扩展（Extensions）→ 安装扩展（Install Extension），粘贴本仓库地址：

```
https://github.com/Mshiyuan/book-excerpt
```

## 基础操作

**划线**
选中文字后浮动栏弹出，点击「划线」。样式可选下划线 / 波浪线 / 荧光笔，颜色可选当前色系的 5 色推荐或自定义色环，支持跨段落选区。

**想法**
浮动栏点击「写想法」。一条划线可挂多条想法；仅写想法不画线时显示为虚线，与正式划线区分。

**改样式**
点击已有划线，弹出工具栏，即时切换样式 / 颜色，或执行 复制 / 删除 / 加想法 / 查看想法 / 做书摘 / 加入合并。

**划线合并**
设置里开启「划线合并」后，点已有划线的工具栏上会多一个「加入合并」；勾好几条后通过悬浮篮子或笔记本的「完成」触发合并，选择存为笔记本条目还是直接生成书摘卡片，也可以选择是否删除原来的散乱划线。合并出的书摘正文可直接点击修改。

**划线持久化**
通过事件订阅 + MutationObserver 双重还原，刷新页面后自动恢复。

## 书摘卡片

入口：浮动栏「书摘」按钮 / 想法编辑器「做成书摘」 / 笔记卡片分享图标 / 想法列表抽屉分享图标。

- **排版**（8 套）：经典、人像、横幅、磁带、影帧、诗笺、锦书、日历
- **颜色**：17 个预设 + 自定义（按背景明度自动配深/浅字色，可单独指定字色）
- **字体**（6 种）：思源宋体、京华老宋体、霞鹜文楷、仓耳今楷、思源黑体、平方韶华
- **排版微调**：正文字号、行距、字距、整体宽度均可调
- **出处**：用户名 / 作者 / 书名 / 章名，均可在「编辑出处」自定义
- **头像**：用户头像 / 角色头像 / 自定义上传
- **水印**：默认显示 SillyTavern，可自定义文字或关闭
- **正文打码**：分享时一键隐去角色名 / 用户名，形式可选涂黑 / 符号 / 自定义符号
- **保留原文删除线**：书摘卡片可保留被划掉文字的删除线效果
- **保存方式**：下载文件 / 弹图长按（内嵌浏览器下载没反应时的自救方式），下载失败自动改用弹图

## 划线色系

工具栏与笔记筛选中的 5 个推荐颜色由色系决定：莫兰迪、马卡龙（默认）、蒙德里安、孟菲斯、马蒂斯。非推荐色可通过工具栏的彩虹按钮调出系统色环选择。

## 自定义模板

入口：设置 - 模板栅格 - 「+ 导入」，支持 `.json`（`{ "name": "模板名", "css": "...CSS..." }`）、`.css` 或直接粘贴 CSS 文本。CSS 选择器使用 `.be-card.be-custom`，可用类名见 `index.js` 内注释。每个自定义模板支持导出为 JSON 分享和删除。

## 笔记本

入口：SillyTavern 扩展菜单 - 书摘笔记。按角色聚合展示划线与想法，支持搜索、筛选（类型 / 划线样式 / 颜色）、想法列表查看，以及划线合并的勾选批量操作。

## 数据

设置 - 数据：导出全部（JSON） / 导入（合并 JSON） / 清空。

存储位置：浏览器 localStorage
- `book-excerpt:settings` 设置项
- `book-excerpt:notes` 笔记数据（按角色索引）

## 实现细节

- 截图库：html2canvas 1.4.1，随仓库分发在 `lib/` 目录，加载时本地文件优先，4 个 CDN 镜像（staticfile / bootcdn / jsdelivr / unpkg）作为兜底
- 卡片渲染：clone 到隔离 iframe 后再截图，避免扫描主文档样式表；头像预处理为高分辨率方形 PNG
- pixelRatio 限制 2x，canvas 用后释放，避免移动端 OOM
